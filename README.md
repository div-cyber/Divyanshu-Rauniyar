## 🛠️ Tech Stack

This portfolio is built with a modern and robust tech stack to ensure a high-quality user experience and development workflow.

- **Framework:** [React](https://reactjs.org/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **UI Components:** [Shadcn UI](https://ui.shadcn.com/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Testing:** [Vitest](https://vitest.dev/)

## ✨ Features

This portfolio showcases a variety of features and components, including:

- **Responsive Design:** Fully responsive layout that looks great on all devices.
- **Interactive UI:** Engaging user interface with smooth animations and transitions.
- **Component-Based Architecture:** A modular and scalable codebase built with reusable components.
- **Contact Form:** A functional contact form using EmailJS for seamless communication.
- **Particle Background:** An interactive particle background to create a dynamic and engaging visual experience.

## 📂 Project Structure

The project is organized into the following directories:

- **src/components:** Contains all the reusable UI components.
- **src/pages:** Contains the main pages of the application.
- **src/data:** Contains the portfolio data, such as project details and skills.
- **src/assets:** Contains all the static assets, such as images and fonts.

## ⚙️ Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Make sure you have Node.js and npm installed on your machine.

- [Node.js](https://nodejs.org/)
- [npm](https://www.npmjs.com/get-npm)

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/your-username/your-repo-name.git
    ```
2.  Install NPM packages
    ```sh
    npm install
    ```
3.  Start the development server
    ```sh
    npm run dev
